API Reference

Connect your agent code to the sandbox for integration testing. All endpoints are CORS-enabled.

MethodPathDescription
GET /api/sandbox/scenarios List every scenario (seeded + user-created).
POST /api/sandbox/scenarios Create a user scenario. Body: name, template, personas, description?, agent?, openingMessage?. Returns id + demo URL.
GET /api/sandbox/scenarios/:id Get one scenario definition.
DELETE /api/sandbox/scenarios/:id Delete a user scenario. Requires X-Dev-Password header.
GET /api/sandbox/scenarios/:id/state Read the current mutable mock state (timesheets, absences, etc.).
POST /api/sandbox/scenarios/:id/state Merge updates into the scenario state.
DELETE /api/sandbox/scenarios/:id/state Reset state back to the scenario initial values.
GET /api/sandbox/scenarios/:id/messages Fetch chat history. Use ?since=timestamp to poll for new messages.
POST /api/sandbox/scenarios/:id/messages Post a message into the scenario chat from a persona.
POST /api/sandbox/scenarios/:id/configure AI-assisted scenario configuration. Body: context. Returns suggested system prompt, opening message, and persona config. Requires X-Dev-Password.
POST /api/sandbox/scenarios/:id/analyse Analyse scenario performance and suggest improvements. Requires X-Dev-Password.
POST /api/sandbox/webhook External agent → sandbox. Posts a message and optionally mutates state in one call.
// Example: your agent posts a nudge into the sandbox const res = await fetch('/api/sandbox/webhook', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ scenarioId: 'timesheet-chaser', agentName: 'timesheet-agent', text: 'Hi Alice — your timesheet is missing Thu and Fri. Need a hand?', actions: [ { label: 'Open Workday', action: 'open_timesheet' }, { label: 'Dismiss', action: 'dismiss' } ], stateUpdates: { escalation: { nudgeCount: 1 } } }) });
